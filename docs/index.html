Features

Mirror Gold is a doubles-focused QoL and Difficulty Hack of Pokémon HeartGold, built using hg-engine. All non-legendary/mythical Pokémon Gens 1-9 and Megas are available throughout the game both in the wild and featured in hand-crafted trainer battles. Weaker Pokémon have received buffs to bring power levels in line and balance things across the board.

TL;DR Changes

  • All trainer battles are Double Battles!
  • Non-legendary Pokémon Gens 1-9 are available in the game and fully rebalanced.
  • Mega Evolution (check Mahogany Town)
  • Portable PC with R button
  • Optional Permadeath/Hardcore mode (currently separate xdelta patch)
  • Free healing items on visiting a Pokécenter to reduce time spent there
  • Rage Candy Bar guy no longer stops you from going towards Route 44
  • Forgettable HMs and Re-usable TMs + many more QoL features

Move Changes

  • Re-usable TMs
  • Forgettable HMs
  • Many moves have been rebalanced. Typically reducing RNG elements and equalizing power a bit
  • Evasion increases are no longer present in the game (though accuracy drops still exist)

Trainer Changes

  • Fully hand-curated trainer shakeups to introduce more challenge and variety
  • Refer to the trainer documentation above for specifics!

Pokémon Changes

  • Stats, types, abilities, and role changes for all Pokémon
  • Much better Pokémon availability. Wiki contains location information for each. All should be available before the Elite 4
  • More Eviolite Pokémon are now viable paired with QoL of Eviolite now preventing evolution when held
  • Refer to the Pokédex documentation above for specifics, but every non-legendary Pokémon has been rebalanced. The goal is for all Pokémon to be viable (even outside of Mega Evolution) so many have received significant buffs

Item Changes

  • Held items (except berries) are returned to you at the end of the battle
  • Cannot use healing items in battle (held items still work)
  • Poké Marts are the primary way in which new items are added to this hack so check them out in each town
  • Nature-changing mints are available throughout the game in Poké Marts
  • Eviolite prevents regular evolution

EVs/IVs Changes

  • IVs are "lucky" meaning you will most likely end up with 30 IVs in each stat though 0 and 31 are still possible and as likely as they are in vanilla
  • All Pokémon grant 2 EVs in all stats
  • EV/IV viewer in the Pokémon stats panel in-game by pressing L or R
